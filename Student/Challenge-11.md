## Descrição

Durante este desafio, você aprenderá sobre gerenciamento de pacotes e as atividades mais comuns como atualizar listas de distribuição de pacotes, instalar e desinstalar pacotes em uma distribuição Linux baseada em Debian.

- Atualize as listas de distribuição de pacotes
- Atualize os pacotes instalados na sua máquina virtual
- Instale os seguintes pacotes
    - git
    - php-fpm
    - nginx
- Desinstale o pacote chamado nano
- Mostre os detalhes do pacote vim
- Encontre o pacote para o runtime de container Docker e instale


## Critérios de Sucesso

1. Garanta que as listas de distribuição estão atualizadas
2. Tenha os pacotes instalados atualizados
3. Garanta a instalação dos pacotes git, php-fpm e nginx
4. Certifique-se de que o pacote nano foi removido do sistema
5. Valide que você tem os detalhes do pacote vim
6. Garanta que o runtime de container Docker está instalado
